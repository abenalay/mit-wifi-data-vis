# Demo

Go to [wifi.firebaseapp.com](http://wifi.firebaseapp.com).
# Usage

1. Launch an HTTP Server to serve static files in this directory: `python -m SimpleHTTPServer`

2. Navigate your browser to `http://localhost:8000/`
